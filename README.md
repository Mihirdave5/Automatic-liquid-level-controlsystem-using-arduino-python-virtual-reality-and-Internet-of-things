# Automatic-liquid-level-controlsystem-using-arduino-python-virtual-reality-and-Internet-of-things
The Aim of this system is to control and maintain liquid level such as, “Water”,”Chemical”,”Syrup” etc of vessel to avoid wastage and also it enables graphical live monitoring of system using python and Internet of Things. System can be monitored from local control room and also through Internet by accessing registered webpage of system.it also include “Dry run” Protection  to protect pump from damage system has low level ,medium level, high level, dry run indicators and one alert buzzer.
